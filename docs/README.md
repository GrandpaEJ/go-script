## üöÄ Quick Start

1. **Install Go-Script**:
   ```bash
   go install github.com/GrandpaEJ/go-script/cmd/gos@latest
   ```

2. **Create your first program**:
   ```gos
   func main():
       print("Hello, Go-Script!")
   ```

3. **Run it**:
   ```bash
   gos run hello.gos
   ```
